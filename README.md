# Installation

1. Install via composer:
```sh
composer require fragotesac/laminas-api-tools-documentation-security
```
   
3. Add module to you Laminas Project in config/modules.config.php

```php
<?php

return [
    /**
    Modules
    */
    'LaminasApiToolsDocumentationSecurity' 
];
```
